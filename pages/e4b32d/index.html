    从零开始搭建自己的脚手架

    # 搭建脚手架常用工具库

    名称 功能
    commander (opens new window) 命令行自定义指令
    inquirer (opens new window) 命令行询问用户问题，记录回答结果控制台输出
    chalk (opens new window) 内容样式美化
    ora (opens new window) 控制台 loading 样式
    figlet (opens new window) 控制台打印 logo
    easy-table (opens new window) 控制台输出表格
    download-git-repo (opens new window) 下载远程模版
    fs-extra (opens new window) 系统fs模块的扩展，提供了更多便利的 API，并继承了fs模块的 API
    cross-spawn (opens new window) 支持跨平台调用系统上的命令

    注意

    重新执行npm link之前需要在本地node安装环境中node_global/node_modules下删除连接的三个执行文件

    否找只是执行npm link 并不会覆盖之前的npm link 实测
